AI Studio

Видео, фото и анимация из одного экрана

Интерфейс теперь поддерживает три режима: генерация видео по тексту, генерация изображения по тексту и анимация загруженного фото в видео. Для видео используется модель Replicate, которая реально поддерживает длительность до 12 секунд, поэтому выбор 10 секунд больше не теряется.

3 режима text-to-video, text-to-image, image-to-video
3-12 сек корректная передача длительности в API
720p быстрое видео с нормальным качеством

Что изменено

Старый ключ больше не должен храниться в коде. Сервер теперь читает только `REPLICATE_API_TOKEN` из переменных окружения Vercel.

Как сделать ролики интереснее

К каждому промпту автоматически добавляются кинематографичные детали: движение камеры, свет, среда и цветокоррекция. Это повышает визуальную выразительность без усложнения интерфейса.

Фото в видео

Можно загрузить изображение, и оно уйдет в image-to-video как первый кадр. На клиенте картинка автоматически сжимается перед отправкой.
